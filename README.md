# AI Marketing Compliance Bot

An AI-powered system that evaluates marketing content against Canadian advertising regulations using a Retrieval-Augmented Generation (RAG) architecture.

Developed as a project in the Schulich Master of Management in Artificial Intelligence (MMAI) program in collaboration with the Future of Marketing Institute (FMI).
